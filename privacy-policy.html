LineOff 개인정보 처리방침

시행일: 2026년 4월 20일

LineOff(이하 "앱")는 사용자의 개인정보를 소중히 다루며, 관련 법령 및 Apple App Store·Google Play 정책을 준수합니다. 본 방침은 앱이 수집·이용·처리위탁하는 개인정보의 범위와 목적을 안내합니다.

1. 수집하는 항목

구분항목수집 경로
계정 정보이메일 주소, 로그인 제공자 식별자(Apple/Google/Kakao)회원가입·로그인 시
사용자 콘텐츠대본 텍스트, 배역 정보, 녹음 텍스트, 연습 기록, 사용자가 작성한 설정앱 사용 중
기기·사용 정보기기 식별자, 언어 설정, 앱 버전, 구독 상태자동 수집
결제 정보영수증(Apple/Google), 구독 상품 ID, 트랜잭션 ID앱 내 구입 시
마이크·카메라 입력음성 인식 결과(텍스트), 촬영한 이미지사용자가 기능 실행 시에만

2. 수집 및 이용 목적

  • 회원 인증 및 계정 관리
  • 대본 저장·동기화 및 연습 기능 제공
  • AI 기반 분석(배역 추출, 장면 분할, 연기 힌트 생성)
  • 음성 합성(TTS) 및 음성 인식을 통한 훈련 지원
  • 구독 결제·검증 및 플랜별 기능 제공
  • 푸시 알림(연습 리마인더, 연속일 응원)
  • 앱 품질 개선 및 오류 분석

3. 제3자 서비스에 대한 처리위탁

앱은 다음 서비스 제공자에게 데이터 처리를 위탁합니다. 각 제공자는 자체 개인정보 처리방침에 따라 데이터를 처리합니다.

수탁자위탁 업무전송 데이터
Supabase Inc. (미국) 인증·데이터베이스·파일 저장 계정 정보, 사용자 콘텐츠, 구독 상태
Anthropic PBC (미국) Claude AI — 배역 추출, 장면 분할, 연기 힌트 생성, PDF 텍스트 추출 대본 텍스트(이미지 포함 시 이미지)
Google LLC (미국) Cloud Text-to-Speech — 대본 음성 합성 음성 합성 대상 텍스트
Google LLC — AdMob 광고 제공 기기 광고 식별자, 사용자 ATT 동의 상태
Apple Inc. / Google LLC 앱 내 구입 결제·영수증 검증 영수증, 트랜잭션 ID
Expo (EAS) 푸시 알림 전송 푸시 토큰

AI 기능(Anthropic, Google Cloud TTS)에 전송된 대본·텍스트는 모델 학습 용도로 사용되지 않습니다(각 제공자의 상용 API 약관에 따름).

4. 보유 및 파기

  • 계정·사용자 콘텐츠: 회원 탈퇴 시 즉시 파기
  • 탈퇴 사유: 서비스 개선 목적으로 이메일 주소와 사유가 익명화되어 보존될 수 있음
  • 결제·영수증 정보: 관련 법령에 따라 5년간 보관(전자상거래법)
  • 앱 내 로컬 데이터(AsyncStorage): 앱 삭제 시 함께 제거

5. 이용자의 권리

  • 접근·열람: 앱 내 설정에서 저장된 데이터 확인 가능
  • 수정: 앱 내에서 직접 수정
  • 삭제: 앱 내 마이페이지 → "계정 삭제" 버튼으로 즉시 탈퇴 및 데이터 파기
  • 동의 철회: 권한 설정(마이크·카메라·알림·추적)은 기기 설정에서 개별 철회 가능
  • 문의: 아래 연락처

6. 아동의 개인정보

LineOff는 만 13세 미만(EU/영국 기준 만 16세 미만) 아동을 주요 대상으로 하지 않으며, 의도적으로 해당 연령의 개인정보를 수집하지 않습니다. 보호자가 아동의 계정 생성 사실을 발견하신 경우 아래 연락처로 문의해 주시면 즉시 삭제해드립니다.

7. 보안 조치

  • 전송 구간 암호화(HTTPS/TLS)
  • Supabase Row Level Security로 사용자별 접근 제한
  • 비밀번호 저장 없음(OAuth 기반 인증)

8. 방침 변경

본 방침이 변경될 경우 앱 내 공지 또는 본 페이지를 통해 안내합니다. 중요한 변경은 시행일 7일 전까지 고지합니다.
